Setup¶

This notebook requires some packages besides pytorch-lightning.

[1]:

! pip install --quiet "pytorch-lightning>=1.4" "matplotlib" "ipython[notebook]" "torch>=1.6, <1.9" "torchmetrics>=0.6"

Create a Markdown cell¶

# %% [markdown]

the content of single cell shall be connected with # at each line, so for example: # Add some text that will be rendered as markdown text.

Create a code cell¶

# %%

[2]:

import torch

print(torch.__version__)

1.8.1+cu102

Add any Python codes¶

Easy integration with Python ecosystem libraries component.

For example create a simple plot with matplotlib with an image:

From: https://matplotlib.org/stable/gallery/lines_bars_and_markers/simple_plot.html

[3]:

import matplotlib.pyplot as plt  # noqa: E402
import numpy as np  # noqa: E402

# Data for plotting
t = np.arange(0.0, 2.0, 0.01)
s = 1 + np.sin(2 * np.pi * t)

fig, ax = plt.subplots()
ax.plot(t, s)

ax.set(xlabel="time (s)", ylabel="voltage (mV)", title="About as simple as it gets, folks")
ax.grid()

fig.savefig("test.png")
plt.show()
